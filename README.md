# Demand Forecasting Model (Machine Learning)

##  Overview
This project predicts product demand using time-series forecasting techniques.

## Tools Used
- Python (Pandas, NumPy)
- Facebook Prophet
- Matplotlib

## Process
1. Data cleaning and preprocessing  
2. Feature extraction (trend, seasonality)  
3. Model training using Prophet  
4. Evaluation using MAE, MAPE, RMSE  

## 📊 Results
- Improved forecasting accuracy compared to baseline methods  

## 🚀 Key Learning
This project strengthened my understanding of time-series modelling and data preparation workflows.
